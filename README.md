## ✨ Features

### 💬 Chat Interface
- Clean chat UI — user messages **right**, AI messages **left**
- **Streaming responses** — text appears word by word like ChatGPT
- **Typing indicator** (3 bouncing dots) while AI responds
- Auto-scroll to latest message
- **Timestamp** on every message
- **Copy button** on hover for each message

### 🧠 Powered by Claude API
- Model: **claude-sonnet-4-6** (latest Anthropic model)
- Full **conversation memory** — context sent with every request
- 3 personality modes: **Friendly / Professional / Concise**
- Friendly error messages when API fails

### 📁 Chat History
- **Multiple conversations** saved in sidebar
- **New Chat** button — starts fresh
- **Rename** chats (double-click or click ✏️)
- **Delete** chats with confirmation
- All data saved in **LocalStorage** — no backend needed

### 🎨 Themes & Customization
- 🌙 **Dark Mode default** + ☀️ Light Mode toggle
- 3 **accent colors**: Blue / Purple / Green
- 3 **font sizes**: Small / Medium / Large

### ⚡ Quick Prompts
```
👨‍💻 Explain Code
✉️  Write Cover Letter
🐛  Debug JavaScript
📄  Resume Review
💡  Project Ideas
📚  Learning Resources
```

### 📊 Stats Panel
- Total messages sent
- Total chats created
- Most used topic (keyword analysis)
- Total words generated by AI

### ⚙️ Settings Panel
- Add/change Anthropic API key
- Switch AI personality
- Change font size
- Clear all history

## 🛠️ Tech Stack

| Tech | Purpose |
|------|---------|
| **HTML5** | Structure |
| **CSS3** | Dark/light theme, glassmorphism, animations |
| **Vanilla JavaScript** | All logic — no frameworks |
| **Claude API (Anthropic)** | AI responses (claude-sonnet-4-6) |
| **Marked.js** | Markdown rendering in AI responses |
| **Prism.js** | Syntax highlighted code blocks |
| **Vercel Edge Functions** | API proxy (keeps API key secure) |
| **LocalStorage** | Chat history & settings |

## 🚀 Deploy Your Own

### 1. Fork & Clone
```bash
git clone https://github.com/noorahmadstudent2024-sketch/noorai.git
cd noorai
```

### 2. Deploy to Vercel
```bash
npx vercel --prod
```

### 3. Add Environment Variable
In Vercel dashboard → Settings → Environment Variables:
```
ANTHROPIC_API_KEY = sk-ant-your-key-here
```

### 4. OR use the Settings Panel
Open the live site → click **Settings** (bottom of sidebar) → paste your API key.

Get your API key at [console.anthropic.com](https://console.anthropic.com)
